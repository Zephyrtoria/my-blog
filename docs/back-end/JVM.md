# JVM

## 概述

Java Virtual Machine Java程序（二进制字节码）运行环境

- 一次编写，到处运行
- 自动内存管理，垃圾回收功能
- 数组越界检查
- 多态

JVM + 基础类库 = JRE

JRE + 编译工具 = JDK：Java开发工具包
